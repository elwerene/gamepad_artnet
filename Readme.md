Receive gamepad events and forward via Artnet. Can be used to control QlcPlus and others.

## Installation

```
cargo install gamepad_artnet
```

## Usage

```
gamepad_artnet 

USAGE:
    gamepad_artnet <ARTNET_HOST> <ARTNET_UNIVERSE>

ARGS:
    <ARTNET_HOST>        
    <ARTNET_UNIVERSE>    

OPTIONS:
    -h, --help    Print help information
```
